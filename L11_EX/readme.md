# Sum Calculator Navigation App

## Description

An Android application built with Kotlin and Jetpack Compose demonstrating multi-screen navigation and data passing between screens.

## Features

- **Screen 1:** User inputs their name.
- **Screen 2:** User inputs two integers. Includes Back navigation.
- **Screen 3:** Displays the user's name and the calculated sum of the inputs. Includes navigation to return Home (reset) or go Back.

## Technologies

- Kotlin
- Jetpack Compose
- Android Navigation Compose
- Material Design 3
